# 全国茶叶批发市场价格与交易量分析看板 - 部署说明文档

## 一、项目概述

本项目是一个基于 Streamlit 的全国茶叶批发市场价格与交易量分析看板，提供茶叶市场数据的可视化分析功能。

### 主要功能
- 📈 **价格趋势分析**：K线图展示各品种价格走势，支持30天、90天、一年时间维度
- 🏪 **市场分布**：各产区交易量占比饼图，茶叶上市时间轴
- 🌸 **明前/雨前对比**：明前茶与雨前茶价格差异对比分析
- 🌤️ **天气影响分析**：产区天气预警集成，灾害性天气标注
- 📊 **数据总览**：交易明细与统计汇总

## 六、部署到生产环境

### 方式一：Streamlit Community Cloud（推荐）

1. 将代码推送到 GitHub 仓库
2. 访问 [Streamlit Cloud](https://streamlit.io/cloud)
3. 点击 "New app"，选择你的仓库和分支
4. 设置主文件路径为 `app.py`
5. 点击 "Deploy" 完成部署

### 方式二：Docker 部署

创建 `Dockerfile`：
```dockerfile
FROM python:3.9-slim

WORKDIR /app

COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY . .

EXPOSE 8501

CMD ["streamlit", "run", "app.py", "--server.port=8501", "--server.address=0.0.0.0"]
```

构建并运行：
```bash
docker build -t tea-dashboard .
docker run -p 8501:8501 tea-dashboard
```

### 方式三：Nginx 反向代理

Nginx 配置示例：
```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://127.0.0.1:8501;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;

        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
```

## 八、功能使用说明

### 1. 侧边栏筛选
- **选择茶叶品种**：下拉选择要分析的具体品种
- **选择交易市场**：可查看单个市场或全部市场数据
- **选择日期范围**：自定义分析时间区间

### 2. 价格趋势分析
- 切换时间周期：近30天/90天/一年
- 查看K线图，支持缩放、悬停查看详情
- 异常价格波动会以⭐标记并在右侧预警区显示

### 3. 市场分布
- 查看各产区交易量占比饼图
- 查看各品种交易量排行
- 查看春茶/夏茶/秋茶上市时间轴

### 4. 明前/雨前对比
- 选择具体品种或全部品种
- 查看价格差异百分比
- 横向对比各品种的明前/雨前价格

### 5. 天气影响分析
- 选择产区查看天气趋势
- 查看灾害性天气预警（霜冻、洪涝、高温等）
- 了解天气对茶叶产量和价格的潜在影响

## 十、常见问题

### Q1: 如何更新真实数据？
A: 替换 `data/tea_trading_data.csv` 和 `data/weather_data.csv` 为真实数据，保持相同的字段结构即可。

### Q2: 如何添加新的茶叶品种？
A: 在 `generate_data.py` 的 `TEA_VARIETIES` 列表中添加品种名称，并配置对应的基准价格和交易量。

### Q3: 如何调整价格异常检测阈值？
A: 在 `data_loader.py` 的 `detect_price_anomalies` 函数中修改 `threshold` 参数（默认为0.15，即15%）。

### Q4: 应用运行缓慢怎么办？
A: 
- 增加缓存时间：调整 `@st.cache_data(ttl=3600)` 的值
- 减少数据量：缩短模拟数据的时间范围
- 使用更快的存储：将数据文件放在SSD上

### Q5: 如何自定义主题颜色？
A: 在 `.streamlit/config.toml` 中修改主题配置，或在 `app.py` 中调整CSS样式。
